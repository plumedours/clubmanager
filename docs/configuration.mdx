# Première configuration

L’assistant de configuration permet de préparer l’application selon les informations de votre organisation.

## Informations recommandées

- Nom du club ou de l’association
- Adresse
- Téléphone
- Courriel
- Logo
- Préférences d’affichage
- Informations utilisées dans les rapports
