# Paramètres - E-mail

Permet de configurer l’envoi d’e-mails depuis l’application.

<img src="/clubmanager/img/docs/parametres/02.png" alt="Paramètres Email" />

---

## Configuration SMTP

- Serveur SMTP
- Port
- Adresse expéditeur
- Nom expéditeur
- Identifiants (utilisateur / mot de passe)
- SSL / TLS

---

## Exemple (Gmail)

- Serveur : smtp.gmail.com
- Port : 587
- SSL/TLS : activé

---

## Cartes de membre

Vous pouvez personnaliser :

- Sujet du message
- Contenu du message

👉 Les variables dynamiques sont supportées (ex : `{MemberName}`)

---

## ⚠️ Important

- Vérifiez vos identifiants SMTP
- Testez l’envoi après configuration
