# Paramètres - Punch

Configure le fonctionnement du système de pointage.

<img src="/clubmanager/img/docs/parametres/04.png" alt="Paramètres Punch" />

---

## Options

### Confirmation
Active ou désactive la confirmation avant enregistrement d’un punch.

### Anti doublon (minutes)
Empêche les enregistrements multiples rapprochés.

👉 Si un punch existe déjà dans ce délai, une alerte s’affiche.

---

## 💡 Exemple

- Valeur : 5 minutes  
→ Empêche plusieurs scans accidentels

---

## Astuce

Désactivez la confirmation pour un usage rapide avec QR code.
