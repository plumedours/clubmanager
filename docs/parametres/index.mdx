# Paramètres

La section **Paramètres** permet de configurer entièrement l’application selon les besoins de votre organisation.

<img src="/clubmanager/img/docs/parametres/01.png" alt="Paramètres ClubManager" />

---

## ⚙️ Organisation des paramètres

Les paramètres sont regroupés en plusieurs onglets :

- 🏢 Club → Informations générales et branding
- 📧 E-mail → Configuration SMTP et envois
- 🖥️ Application → Comportement global
- ⏱️ Punch → Gestion des présences
- 🛠️ Maintenance → Réinitialisation et configuration
- 🔒 Sécurité → Protection de l’application

---

## 💾 Sauvegarde

- Cliquez sur **Enregistrer** pour appliquer les modifications
- Cliquez sur **Recharger** pour annuler les changements

👉 Les paramètres sont appliqués immédiatement après enregistrement.

---

## 💡 Bonnes pratiques

- Configurez les paramètres dès la première utilisation
- Testez les e-mails après configuration SMTP
- Activez la sécurité pour protéger les données sensibles
