# Paramètres - Maintenance

Permet de gérer les opérations avancées.

<img src="/clubmanager/img/docs/parametres/05.png" alt="Paramètres Maintenance" />

---

## Assistant de configuration

Permet de :
- Modifier la connexion à la base de données
- Reconfigurer l’application

---

## ⚠️ Zone dangereuse

Permet de réinitialiser complètement la base :

- Membres
- Présences
- Transactions
- Catégories

---

## Sécurité

- Confirmation obligatoire
- Saisie du mot RESET

---

## 🔒 Important

Une sauvegarde automatique est créée avant la réinitialisation.
