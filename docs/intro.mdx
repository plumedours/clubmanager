# Présentation

ClubManager est une application de gestion conçue pour les clubs, associations et petites organisations.

Elle permet de centraliser les membres, les transactions, les présences, les rapports et les paramètres de l’organisation.

<img src="/clubmanager/img/docs/tableau-de-bord/01.png" alt="Tableau de bord ClubManager" />

## Ajouter des images dans la documentation

Toutes les captures sont placées dans le dossier :

```txt
static/img/docs/
```

Dans un fichier de documentation, ajoute une image avec cette syntaxe :

```md
<img src="/clubmanager/img/docs/membres/01.png" alt="Description de l’image" />
```

Pour GitHub Pages, on conserve le préfixe `/clubmanager/`.
