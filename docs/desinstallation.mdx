# Désinstallation

Cette page explique comment supprimer ClubManager de votre ordinateur.

---

## 🗑️ Méthode standard

1. Ouvrez le menu **Paramètres Windows**
2. Allez dans **Applications**
3. Recherchez **ClubManager**
4. Cliquez sur **Désinstaller**

---

## ⚙️ Via le panneau de configuration

1. Ouvrez le **Panneau de configuration**
2. Cliquez sur **Programmes et fonctionnalités**
3. Sélectionnez **ClubManager**
4. Cliquez sur **Désinstaller**

Selon la configuration, certaines données peuvent rester sur votre ordinateur.

Si nécessaire, contactez le support pour savoir comment les supprimer complètement.
