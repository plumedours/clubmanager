# Installation

Cette page explique comment installer ClubManager à partir des fichiers transmis après une contribution libre.

## Étapes générales

1. Lancez le fichier d’installation fourni.
2. Suivez les étapes de l’assistant.
3. Ouvrez ClubManager.
4. Complétez la première configuration.
