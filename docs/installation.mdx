# Installation

Cette page explique comment installer ClubManager à partir des fichiers transmis après une contribution à prix libre.

---

## 💻 Prérequis

Avant de commencer, assurez-vous de disposer :

- d’un ordinateur sous Windows ;
- des droits nécessaires pour installer une application ;
- du fichier d’installation fourni.

---

## ⚙️ Installation de l’application

Suivez les étapes ci-dessous :

### 1. Lancer l’installation

Double-cliquez sur le fichier d’installation fourni.

### 5. Premier lancement

Ouvrez ClubManager depuis le menu démarrer ou le raccourci créé.

---

## 🚀 Première configuration

Lors du premier lancement, il vous sera demandé de configurer :

- les informations de votre organisation ;
- les paramètres de base ;
- les préférences de l’application.

👉 Consultez la section suivante :

👉 [Première configuration](/clubmanager/documentation/configuration)

---

## ⚠️ Problèmes fréquents

### L’installation ne démarre pas

- Vérifiez que le fichier est bien téléchargé
- Essayez un clic droit → **Exécuter en tant qu’administrateur**

### Message de sécurité Windows

Windows peut afficher un avertissement :

- cliquez sur **Informations complémentaires**
- puis **Exécuter quand même**
