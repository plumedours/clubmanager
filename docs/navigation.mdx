# Navigation dans l’application

Cette page présente les principaux menus de ClubManager et leur utilité.

L’application est organisée autour d’un menu principal qui donne accès aux différentes fonctionnalités.

---

## 🧭 Menu principal

<img src="/clubmanager/img/docs/interface/menu/01.png" alt="Menu principal ClubManager" />

Le menu principal permet d’accéder rapidement aux différentes sections de l’application :

- Fichier  
- Membres  
- Présences  
- Finances  
- Rapports  
- Paramètres  
- Aide  

---

## 📁 Menu Fichier

<img src="/clubmanager/img/docs/interface/menu/02.png" alt="Menu Fichier" />

Options disponibles :

- **Tableau de bord**  
  Retour à l’écran principal de l’application

- **Quitter**  
  Ferme l’application

---

## 👥 Menu Membres

<img src="/clubmanager/img/docs/interface/menu/03.png" alt="Menu Membres" />

Options disponibles :

- **Liste des membres**  
  Affiche tous les membres enregistrés

- **Nouveau membre**  
  Permet d’ajouter une nouvelle fiche membre

---

## ⏱️ Menu Présences

<img src="/clubmanager/img/docs/interface/menu/04.png" alt="Menu Présences" />

Option disponible :

- **Punch / Accueil**  
  Permet d’enregistrer les passages ou présences

---

## 💰 Menu Finances

<img src="/clubmanager/img/docs/interface/menu/05.png" alt="Menu Finances" />

Options disponibles :

- **Transactions**  
  Affiche la liste des transactions

- **Nouvelle transaction**  
  Permet d’ajouter une transaction

- **Catégories**  
  Permet de gérer les catégories financières

---

## 📊 Menu Rapports

<img src="/clubmanager/img/docs/interface/menu/06.png" alt="Menu Rapports" />

Option disponible :

- **Rapports**  
  Accès aux différents rapports générés

---

## ⚙️ Menu Paramètres

<img src="/clubmanager/img/docs/interface/menu/07.png" alt="Menu Paramètres" />

Options disponibles :

- **Configuration**  
  Paramètres généraux de l’application

- **Sauvegardes**  
  Gestion des sauvegardes des données

---

## ❓ Menu Aide

<img src="/clubmanager/img/docs/interface/menu/08.png" alt="Menu Aide" />

Option disponible :

- **Documentation**  
- - Lien direct vers la documentation en ligne
- **Support / Dépanange**  
- - Lien direct vers la page de dépannage en ligne
- **Site web**  
- - Lien direct vers la page web de l'application
- **À propos**  
- - Informations sur l’application

---

## 💡 Conseil

Prenez quelques minutes pour explorer les menus afin de vous familiariser avec l’organisation de l’application.
