# Gestion des transactions

La section **Finances** permet de suivre toutes les opérations financières de votre organisation.

Elle centralise les **entrées**, **sorties**, **catégories** et offre une vue claire de votre situation financière.

---

## 📊 Vue générale

<img src="/clubmanager/img/docs/finances/01.png" alt="Module finances ClubManager" />

L’écran principal est composé de :

- un panneau de filtres ;
- un journal des transactions ;
- des actions rapides ;
- un résumé financier.

---

## 🔍 Filtres

Permet de rechercher et filtrer les transactions :

- période (jour, mois, année) ;
- date de référence ;
- recherche texte ;
- type (entrée / sortie) ;
- catégorie ;
- mode de paiement.

👉 Idéal pour analyser vos finances rapidement.

---

## 📄 Journal des transactions

<img src="/clubmanager/img/docs/finances/06.png" alt="Journal des transactions" />

Le tableau affiche :

- Date  
- Type (Entrée / Sortie)  
- Catégorie  
- Montant  
- Mode de paiement  
- Description  
- Référence  
- Notes  

👉 Les couleurs permettent de distinguer rapidement :

- **Entrées (vert)**  
- **Sorties (rouge)**

## ➕ Ajouter une transaction

Cliquez sur :

👉 **Ajouter une transaction**

<img src="/clubmanager/img/docs/finances/02.png" alt="Bouton ajouter transaction" />

### Champs disponibles

- **Date** *(obligatoire)*  
- **Type** *(Entrée ou Sortie)*  
- **Catégorie** *(obligatoire)*  
- **Montant** *(obligatoire)*  
- **Mode de paiement** *(obligatoire)*  
- Description  
- Référence  
- Notes

## 🎯 Cas d'utilisation

ClubManager permet de gérer facilement :

- les cotisations des membres ;
- les paiements ponctuels ;
- les dépenses du club ;
- les revenus divers ;
- le suivi global de la trésorerie.

---

## 💡 Bonnes pratiques

- utiliser des catégories claires ;
- enregistrer chaque transaction dès qu’elle se produit ;
- vérifier régulièrement le résumé financier ;
- utiliser les filtres pour analyser vos données.
