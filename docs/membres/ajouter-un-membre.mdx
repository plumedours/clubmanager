# Ajouter un membre

La gestion des membres permet de centraliser les informations importantes de chaque personne.

<img src="/clubmanager/img/docs/membres/01.png" alt="Liste des membres" />

## Informations possibles

- Nom
- Prénom
- Coordonnées
- Statut
- Type de membre
- Date d’inscription
- Notes internes
