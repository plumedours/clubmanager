# Cartes membres

ClubManager peut être utilisé pour préparer ou gérer des cartes membres selon la configuration choisie.

## Utilisation possible

- Identifier rapidement un membre
- Faciliter l’enregistrement des présences
- Utiliser un code ou QR code
- Uniformiser l’identité du club
