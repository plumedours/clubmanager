# Cartes membres

ClubManager permet de générer automatiquement des cartes membres avec un QR code unique pour chaque personne.

Ces cartes facilitent l’identification et l’utilisation de certaines fonctionnalités comme l’enregistrement des présences.

---

## 🪪 Génération d’une carte

Depuis la gestion des membres :

1. Sélectionnez un membre
2. Cliquez sur **Carte**

<img src="/clubmanager/img/docs/membres/07.png" alt="Carte membre ClubManager" />

---

## 📦 Contenu de la carte

<img src="/clubmanager/img/docs/membres/05.png" alt="Gestion des membres" />

Chaque carte contient :

- le nom du membre ;
- le numéro de membre ;
- la date d’adhésion ;
- une photo (si disponible) ;
- un **QR code unique**.

---

## 🔳 Utilisation du QR code

Le QR code permet d’identifier rapidement un membre dans l’application.

### Utilisation principale

👉 Enregistrement des présences (Punch)

Lorsqu’un QR code est scanné :

- le membre est automatiquement identifié ;
- un enregistrement de présence est créé.

---

## 🖥️ Matériel recommandé

Pour utiliser les cartes avec QR code, vous pouvez utiliser :

### Scanner USB (recommandé)

- scanner compatible **QR code**
- fonctionne comme un clavier (plug & play)
- aucun pilote spécifique requis

👉 Exemple d’utilisation :

1. brancher le scanner USB ;
2. ouvrir l’écran **Punch / Accueil** ;
3. scanner la carte membre ;
4. le membre est automatiquement reconnu.

---

## 🔄 Alternative sans scanner

Il est aussi possible d'entrer manuellement le numéro du membre.

---

## 🎯 Avantages des cartes membres

- identification rapide ;
- réduction des erreurs ;
- gain de temps à l’accueil ;
- expérience plus professionnelle.

---

## ⚠️ Bonnes pratiques

- imprimer les cartes en bonne qualité ;
- éviter de plier ou détériorer le QR code ;
- conserver une taille suffisante pour un scan fiable.
