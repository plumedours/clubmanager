# Rapports

La section **Rapports** permet de générer, visualiser et exporter les données de votre organisation sous forme de rapports clairs et professionnels.

<img src="/clubmanager/img/docs/rapports/01.png" alt="Rapports ClubManager" />

---

## 📊 Types de rapports

Trois catégories de rapports sont disponibles :

### 💰 Finances
- Résumé annuel
- Détail mensuel
- Répartition par catégorie (entrées / sorties)
- Journal des transactions

### 👥 Présences
- Liste des présences par période
- Distinction membres / non-membres
- Historique des passages

### 🧾 Membres
- Liste complète des membres
- Statut (actif / inactif)
- Dates d’adhésion et d’expiration

---

## 🔎 Filtres et périodes

Chaque rapport peut être ajusté selon vos besoins :

- **Année** (finances)
- **Jour / Mois / Année** (présences)
- **Statut** (membres)

👉 Cela permet d’obtenir rapidement des données précises et exploitables.

---

## 📈 Résumés visuels

Selon le type de rapport, ClubManager affiche :

- Des **cartes de résumé** (totaux, soldes, compteurs)
- Des **tableaux détaillés**
- Des regroupements (ex : par mois ou par catégorie)

👉 Ces éléments facilitent la lecture et l’analyse rapide des données.

---

## 📤 Exportation

Les rapports peuvent être exportés en plusieurs formats :

- **Excel (.xlsx)** → pour analyse ou modification
- **PDF (.pdf)** → pour partage ou archivage

<img src="/clubmanager/img/docs/rapports/03.png" alt="Aperçu rapport" />

👉 Les exports respectent le contenu affiché selon les filtres sélectionnés.

---

## 🎨 Personnalisation (PDF)

Les rapports PDF utilisent le **branding configuré dans les paramètres** :

- Nom de l’organisation
- Logo
- Informations personnalisées

👉 Cela permet de générer des documents professionnels prêts à être partagés.

---

## 📁 Accès aux fichiers

Un bouton permet d’ouvrir directement le dossier contenant les exports :

- Accès rapide aux fichiers générés
- Organisation simple des documents

---

## 💡 Bonnes pratiques

- Utilisez les filtres pour cibler vos rapports
- Exportez régulièrement pour conserver un historique
- Privilégiez le PDF pour le partage officiel
- Utilisez Excel pour les analyses avancées

---

## 🧠 À savoir

- Les données affichées sont toujours à jour selon la base de données
- Les rapports reflètent exactement les filtres actifs
- Aucun recalcul manuel n’est nécessaire
