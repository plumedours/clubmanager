# ClubManager - Site web

Site statique Docusaurus + React + Tailwind CSS pour présenter ClubManager et héberger sa documentation.

## Lancer en local

```bash
npm install
npm run start
```

Adresse locale :

```txt
http://localhost:3000/clubmanager/
```

## Captures d’écran

- Accueil : `static/img/screenshots/`
- Documentation : `static/img/docs/`

Dans une page MDX :

```md

```
